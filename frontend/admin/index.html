Thư viện Media (Supabase Storage)

Media được lưu trữ trực tiếp trong Supabase Storage. Xem và quản lý ảnh/video của từng cây trong trang chi tiết cây.

Truy cập Supabase Dashboard: https://supabase.com để quản lý và xem các file phương tiện của bạn.

Import cây bằng file CSV

Phương thức này giúp tự động tạo hàng loạt cây trồng từ file tọa độ CSV. File CSV cần có các cột tiêu đề: STT, E (Kinh độ/Lng), N (Vĩ độ/Lat).
